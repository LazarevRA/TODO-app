# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

1. Описание проекта:  проект представляет собой веб-сервер с REST API с функционалом базового TODO - листа.
TODO - таск менеджер позволяет:
- добавлять, удалять, и редактировать задачи;
- отмечать задачи выполненными;
- задачи могут иметь переодичность по дням, месяцам, годам и неделям;
- задачи с переодичностью автоматически переносятся при выполнении;
- получать список задач;

2. Выполненные задания со звёздочкой
В минимальной комплектации выполнена только реализация алгоритмов для повторения задач
(weekly и monthly repeat) правила пересчета дат описаны в файле pkg/api/repeatrules.go
Другие задания повышенной сложности не выполнялись, но обязательно будут :)

3. Инструкция по запуску локально:

- Go 1.21+
- SQLite driver (например modernc.org/sqlite)
- Порт 7540 должен быть свободен (или переопределён через TODO_PORT)

4. Инструкция по запуску тестов

- порт должен быть 7540 (или такой же, как в переменной TODO_PORT);
- имя базы по умолчанию — scheduler.db.
- Для запуска всех тестов: go test ./tests

В тестах установлена переменная FullNextDate = true, осуществляющая проверку
функционала переноса даты задачи для недельного и месячного повторения задач.


5. Сборка и запуск через Docker
В текущей реализации Docker не используется.
